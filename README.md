# nipkow

## Setup on ubuntu machine

**Step 1:**

```
sudo apt install npm
```

**Step 2:**

```
git clone --single-branch --branch develop https://github.com/xoken/nipkow.git
```

**Step 3:**
cd into the nipkow app's directory.

```
npm install
```

## To launch nipkow

cd into nipkow app's directory and then

```
npm start
```

open new terminal
cd into nipkow app's directory and then

```
cd /lib/nipkow-sdk
```

```
npm start
```

open new terminal
cd into nipkow app's directory and then

```
npm run build
```

```
npm run electron
```
